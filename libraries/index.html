Python Calendaring Ecosystem

Libraries

All packages maintained by this team. Each one is independent, installable from PyPI, and focused on a specific part of the calendaring stack.

Core

icalendar Core

Reads and writes .ics files. Every other library here depends on it. Maintained under the collective/icalendar GitHub org.
recurring-ical-events Core

Give it a date range, get back every event in it. Recurring events, exceptions, overrides. Used internally by python-caldav.

Clients

python-caldav Client

The only actively maintained Python CalDAV client. Sync and async, scheduling, free/busy, JMAP. Tested against Nextcloud, Baikal, Cyrus, SOGo, Fastmail, Google Calendar, and others.
calendaring-client In Development

One API for CalDAV, JMAP, raw .ics files, and web feeds. Point it at a URL and it figures out what it's talking to.

JMAP

calendaring-jmap In Development

The JMAP calendar implementation currently lives in python-caldav: sync and async clients, calendar and event management, incremental sync, task support, and a full iCalendar to JSCalendar conversion layer. Tested against Cyrus IMAP and Stalwart. This is where it moves once extracted.

Tools and applications

plann CLI

CLI and TUI for managing calendars and tasks. Talks directly to your CalDAV server — no local sync, no middleman. Successor to calendar-cli.
open-web-calendar Application

Turns iCalendar feeds into a calendar you can view in any browser. No account, no database, no tracking. Configure with URL parameters and embed anywhere.
icalendar-anonymizer Tool

Scrubs personal data out of .ics files without breaking the structure. Use it as a Python library, CLI, HTTP API, or just the web UI.
python-mergecal Tool

Merges multiple iCalendar feeds into one. Running in production at MergeCal.org and MyHebrewDates.com.
icalendar-searcher Tool

Filter, sort, and expand calendar components on the client side. Handy when your server's query support doesn't cut it. Works well alongside python-caldav.

Integrations

django-icalendar Integration

Django models for calendar data. CalendarModel, EventModel, forms, and admin integration — calendar support that fits naturally into a Django project.
